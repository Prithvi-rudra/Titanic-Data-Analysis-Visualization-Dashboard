# Titanic-Data-Analysis-Visualization-Dashboard
Exploratory Data Analysis (EDA) and visualization dashboard for the Titanic dataset, demonstrating data cleaning, feature engineering, and professional-level data storytelling using Python and Seaborn.

## Overview
This project performs an Exploratory Data Analysis (EDA) on the Titanic dataset. [cite_start]The primary goal was to create a visualization dashboard to uncover patterns related to survival, demographic distribution, and socioeconomic status. [cite: 5, 9, 101]

## Dataset
- **Dataset:** Titanic Dataset (Kaggle)
- **Preprocessing:** Handled missing values (Age, Embarked), dropped irrelevant columns (Cabin), and engineered features (FamilySize, AgeGroup).

## Visualizations
The dashboard includes the following analyses:
1. **Age Distribution:** Histogram showing the demographic spread of passengers.
2. **Survival by Gender:** Bar chart comparing survival rates between sexes.
3. **Fare by Class:** Boxplot showing fare price distributions across classes.
4. **Age vs. Fare:** Scatterplot analyzing the relationship between age, fare, and survival.
5. **Correlation Heatmap:** Matrix visualizing the relationships between numeric variables.

## How to Run
1. Ensure you have Python installed with the necessary libraries: `pandas`, `numpy`, `matplotlib`, and `seaborn`.
2. Clone this repository.
3. Open the `Titanic_Analysis.ipynb` file in Jupyter Notebook or Google Colab.
4. Run all cells to generate the dashboard.
