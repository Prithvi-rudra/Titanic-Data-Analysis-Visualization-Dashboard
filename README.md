# Titanic-Data-Analysis-Visualization-Dashboard
Exploratory Data Analysis (EDA) and visualization dashboard for the Titanic dataset, demonstrating data cleaning, feature engineering, and professional-level data storytelling using Python and Seaborn.

## Overview
This project performs an Exploratory Data Analysis (EDA) on the Titanic dataset. [cite_start]The primary goal was to create a visualization dashboard to uncover patterns related to survival, demographic distribution, and socioeconomic status. [cite: 5, 9, 101]

## Dataset
- [cite_start]**Dataset:** Titanic Dataset (Kaggle) [cite: 94]
- [cite_start]**Preprocessing:** Handled missing values (Age, Embarked), dropped irrelevant columns (Cabin), and engineered features (FamilySize, AgeGroup). [cite: 14, 15]

## Visualizations
The dashboard includes the following analyses:
1. [cite_start]**Age Distribution:** Histogram showing the demographic spread of passengers. [cite: 17]
2. [cite_start]**Survival by Gender:** Bar chart comparing survival rates between sexes. [cite: 18]
3. [cite_start]**Fare by Class:** Boxplot showing fare price distributions across classes. [cite: 19]
4. [cite_start]**Age vs. Fare:** Scatterplot analyzing the relationship between age, fare, and survival. [cite: 20]
5. [cite_start]**Correlation Heatmap:** Matrix visualizing the relationships between numeric variables. [cite: 21]
